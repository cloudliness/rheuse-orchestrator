# Rheuse — Orchestrator Workspace

This repository contains the Rheuse e-commerce starter and the Master Orchestrator agent scaffolding.

Quick start

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build: `npm run build`
4. Lint: `npm run lint`

Notes

- This project is scaffolded by the Orchestrator agent. See `.github/copilot-instructions.md` for workspace conventions.
- To add a remote and push the initial commit, provide a Git remote URL and I'll add it for you.
